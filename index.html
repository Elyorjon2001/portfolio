Men haqimda

Men Laravel frameworkida backend qismini ishlab chiqaman, MySQL orqali bazalarni boshqaraman. Frontend qismni esa HTML va CSS yordamida yarataman yoki tayyor shablonlarni moslashtiraman. Web saytlar uchun autentifikatsiya va ma'lumotlar bazasi bilan ishlashni bajara olaman.

Rus tilini so‘zlashuv darajasida bilaman va ingliz tilida dasturlashga oid terminlarni tushunaman.
